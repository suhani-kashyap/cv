Netflix Clone

This project mimics the user interface and the quick payment subscription offered by the famous streaming service Netflix. The movies and show details are dynamically pulled in through the tmdb API. The project is built in React.js, uses the Google Firestore Database. The payments are integrated using the Stripe Checkout extension on Firebase.

Daily Journal

Daily Journal is a minimalistic and sleek everyday journaling website. It has been created using an Express Server. It uses Node.js, EJS templates. The framework of the website has been written in HTML and styled using CSS. The project handles requests made by the user with the Express Server. It is a great tool for everyday thoughts!

ToDeLoo

ToDeLoo is a To-Do list made in JavaScript using an Express Server. It consists of two types of lists - General and Work. The Work List can be accessed by adding /work at the end of the URL. It is a great tool to stay organized throughout the day!

The Simon Game is a fun project to play alone or with friends. It is a static website project that uses the JavaScript Document-Object Model to access elements and give them functions. I have used the JS library JQuery for this project. I also integrated sound to make it more user-friendly and interactive.
